# Aurora Salon OS

Mini-aplicație mobilă pentru managementul unui salon de înfrumusețare.

## Features

- Dashboard cu KPI-uri zilnice (venit, confirmări, clienți activi)
- Confirmare programări direct pe WhatsApp — un singur tap
- Gestionare clienți cu risc de no-show evidențiat
- Campanii SMS/WhatsApp pentru clienți inactivi
- Dark mode cu toggle glass-style
- Design Apple-like, mobile-first

## Stack

HTML · CSS · JavaScript — zero dependențe, deploy static pe Vercel.

## Local dev

```bash
npx serve .
```

Deschide [http://localhost:3000](http://localhost:3000).
